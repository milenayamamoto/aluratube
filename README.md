# Aluratube

O Aluratube é um projeto realizado durante a imersão React disponível pela Alura.

### Features

- [x] Conexão com Supabase
- [x] Dark/Light theme
- [x] Barra de pesquisa


## 🚀 Instalando o Aluratube

Para instalar o Aluratube:

```
npm install
```
